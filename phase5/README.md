# Phase 5: Presentation Layer

This folder contains the implementation of Phase 5 - the presentation layer for the restaurant recommendation system. Phase 5 is responsible for displaying the top K restaurant recommendations generated by the LLM in Phase 4 in a user-friendly format.

## Features

### Core Functionality
- **Multiple Output Formats**: Console, JSON, and Markdown rendering
- **Restaurant Details Display**: Name, cuisines, rating, cost, location
- **AI Explanations**: Display of LLM-generated recommendation reasons
- **Flexible Rendering**: Configurable width and formatting options

### UX Enhancements
- **Sorting Options**: By rank, rating, cost, or name
- **Filtering Capabilities**: By rating, cost, cuisine, or location
- **Comparison Tools**: Side-by-side restaurant comparison
- **Preference Refinements**: AI-powered suggestions for better results
- **Filter Suggestions**: Context-aware filtering recommendations

## Usage

### Basic Usage
```bash
# Display recommendations in console format
python phase5/run.py --input storage/recommendations.json

# Output to JSON file
python phase5/run.py --format json --output results.json

# Generate Markdown report
python phase5/run.py --format markdown --output report.md
```

### Enhanced Features
```bash
# Sort by rating (high to low)
python phase5/run.py --sort rating_high

# Filter by minimum rating and maximum cost
python phase5/run.py --filter-rating 4.5 --filter-cost 1500

# Compare specific restaurants
python phase5/run.py --compare 0 1 2

# Show preference refinement suggestions
python phase5/run.py --show-suggestions

# Show available filter suggestions
python phase5/run.py --show-filters

# Enhanced output with all features
python phase5/run.py --format enhanced --sort cost_low --filter-rating 4.5 --show-suggestions
```

- Restaurant rankings with visual indicators

### Enhanced Output
Combines all features:
- Applied filters information

- Preference refinement recommendations

## API Reference

### RecommendationRenderer
Main class for rendering recommendations in different formats.

**Methods:**
- `parse_recommendations()`: Parse Phase 4 JSON into display objects
- `render_console()`: Generate console-formatted output
- `render_json()`: Generate JSON output
- `render_markdown()`: Generate Markdown output

### RecommendationEnhancer
Provides UX enhancements for recommendations.

**Methods:**
- `sort_recommendations()`: Sort by various criteria
- `filter_recommendations()`: Apply filters
- `compare_restaurants()`: Side-by-side comparison
- `get_filter_suggestions()`: Context-aware filter suggestions
- `get_preference_refinements()`: AI-powered refinement suggestions

## Integration with Other Phases

- **Input**: Consumes `recommendations.json` from Phase 4
- **Optional Input**: Uses `preferences.json` from Phase 2 for refinements
- **Output**: Formatted recommendations for end-user consumption

## Dependencies

- Python 3.8+
- No external dependencies required (uses only standard library)
- Designed to work with Phase 4 output format

## Examples

### Basic Display
```bash
python phase5/run.py
```
Shows the top 5 recommendations in enhanced console format.

### Budget-Conscious Display
```bash
python phase5/run.py --sort cost_low --filter-cost 1200
```
Shows cheapest options first, limited to ₹1200.

### High-Quality Options
```bash
python phase5/run.py --sort rating_high --filter-rating 4.7 --top-k 3
```
Shows top 3 highest-rated restaurants with 4.7+ rating.

### Comparison Analysis
```bash
python phase5/run.py --compare 0 1 --show-suggestions
```
Compares first two restaurants and shows improvement suggestions.
